
# Bank of America Stock Analysis

## Overview
This project performs advanced stock analysis using historical Bank of America stock data.

It includes:

- Exploratory Data Analysis
- Technical Indicators
- ARIMA Forecasting
- Machine Learning Models
- LSTM Deep Learning Prediction
- Trading Strategy Backtesting
- Risk Metrics (Sharpe Ratio, Drawdown)
- Automatic PDF Reporting
- Streamlit Interactive Dashboard

## Models Implemented

- ARIMA Time Series Forecasting
- RandomForest / XGBoost Regression
- LSTM Deep Learning Model
- Prophet Forecasting (optional)

## Financial Analytics

- Moving Average Trading Strategy
- Portfolio Backtesting
- Sharpe Ratio
- Maximum Drawdown

## Visualizations

The project automatically generates **40+ charts** including:

- price trends
- distributions
- rolling averages
- volatility plots
- strategy performance

## Streamlit Dashboard

Run the interactive dashboard:

```
streamlit run streamlit_app.py
```

## Installation

Install dependencies:

```
pip install -r requirements.txt
```

## Dataset

Place dataset in project folder:

```
bank_of_america_stock_data.csv
```
